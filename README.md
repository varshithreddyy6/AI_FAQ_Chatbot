# AI FAQ Chatbot

An AI-powered chatbot built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Gemini AI API

## Features

- Modern chatbot UI
- FAQ-based responses
- AI-generated responses
- Chat memory
- Responsive design

## Installation

```bash
npm install
```

## Run Project

```bash
npm start
```

Open:
http://localhost:3000
